# 🧠 Prompts

Este projeto tem como objetivo facilitar a criação de prompts eficientes para Inteligência Artificial utilizando o método **ASK**, dividido em quatro etapas:

* **Agent** → Define o papel da IA
* **Study** → Contextualiza e fornece informações relevantes
* **Plan** → Estrutura o raciocínio e a execução
* **Ask** → Define claramente o que deve ser respondido

## 🚀 Objetivo

Ajudar desenvolvedores e usuários a criarem prompts mais claros, estruturados e eficazes, melhorando a qualidade das respostas geradas por IAs.

## 📁 Estrutura do Projeto

```
prompts/
 ├── agent.md
 ├── study.md
 ├── plan.md
 └── ask.md
```

Cada arquivo representa uma etapa do método ASK.


## 🛠️ Tecnologias

* Markdown (.md)
* Visual Studio Code
